SQL Pizza Sales Analysis Project:
Project Overview:
This project analyzes a Pizza Sales dataset using SQL to extract meaningful business insights.
The objective is to understand sales performance, customer behavior, and product trends to help improve business decisions.

Objectives:
Analyze total revenue and total orders
Identify best-selling and least-selling pizzas
Analyze sales by category and size
Find peak order times
Understand monthly sales trends
Calculate average order value

Tools & Technologies Used:
SQL (MySQL)
Aggregate Functions
GROUP BY & ORDER BY
JOIN Operations
Subqueries
Date Functions
